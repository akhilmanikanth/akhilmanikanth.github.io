ℹ️ About DCT’s Universal Tool (Enhanced)

🚀 Re‑engineered by Akhil Manikanth , this enhanced Universal Dashboard takes Quest → Boral conversions and weekly reporting to a new level. It retains the 100% browser‑only architecture—your data never leaves your machine—while adding smarter features and a streamlined interface.

Built with React , TypeScript , Chart.js (4.4.4) and XLSX.js , the new UI uses modern CSS variables and the Inter font for a clean, responsive design. Live analytics now compare Concrete vs Grout and display a rolling export history, so you always know what’s been processed.

New capabilities include site‑specific uploads (Lobs Hole, Polo Flat) , an optional auto‑fill for missing TR source codes , and richer data filtering for temperature reporting. Drag‑and‑drop uploads , coloured status tags, and confetti‑powered celebrations make day‑to‑day use both efficient and satisfying.

In short, it’s faster, clearer and more powerful—while still delivering lightning‑fast conversions, intelligent weekly reports and advanced data mapping. Enjoy speed , elegance and efficiency , with zero compromise on privacy.
